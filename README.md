# 🎯 Reaction Time Game (Keypress Attention Game)

## 📌 Project Description

This is a simple Reaction Time Game developed for HCI Lab.
It measures how quickly a user responds to visual stimuli (letters and numbers) using keyboard input.

The game evaluates user attention and reaction speed.

---

## 🎮 How to Play

1. Click the **Start** button
2. A random character will appear:

   * 🔤 Letter (A–Z)
   * 🔢 Number (0–9)
3. Press:

   * **A** → if it is a Letter
   * **L** → if it is a Number
4. The game will show:

   * ✅ Correct or ❌ Wrong
   * ⏱ Reaction time in milliseconds

---

## ⚙️ Features

* 🎯 Random character generation
